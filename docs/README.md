# Documentation Index

This repository uses a layered documentation model:

- [AGENTS.md](../AGENTS.md): short contributor entrypoint.
- [contributor/workflow.md](contributor/workflow.md): current build, test, and verification commands.
- [contributor/rules.md](contributor/rules.md): repository-specific conventions and hard rules.
- [architecture/overview.md](architecture/overview.md): module boundaries and design principles.
- [history/vibe/](history/vibe): raw archive migrated from the old `.vibe/` folder.

## Status of Archived Vibe Notes

Files in `docs/history/vibe/` are preserved for traceability. Some commands in that archive are stale (for example references to `just run`). Use documents in `docs/contributor/` and `AGENTS.md` as the source of truth for current workflows.
